## 📌 Overview

This project is an interactive **Amazon Sales Dashboard** built using **Power BI**. It transforms raw sales data into meaningful business insights through dynamic KPIs, interactive filters, and visually appealing charts. The dashboard helps analyze revenue, profit, sales performance, and regional trends for better decision-making.


## 🚀 Dashboard Features

- 📊 Interactive KPI Cards
- 📅 Dynamic Date Filter
- 🌍 Country-wise Sales Analysis
- 🌎 Region-wise Profit Analysis
- 📈 Revenue Trend Analysis
- 🛍️ Sales Channel Distribution
- 📦 Item Type Performance
- 🗺️ Geographic Sales Map
- 🌳 Revenue Distribution Treemap
- 🔍 Interactive Slicers



## 📊 Key KPIs

- 💰 Total Revenue
- 📈 Total Profit
- 💳 Total Cost
- 📦 Total Units Sold
- 🛒 Total Orders
